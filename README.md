# SDKSMS Demo SDKs

This repository contains example code for integrating with the [SDKSMS](https://sdksms.com) API in various programming languages.

## Languages Included

- **PHP**: `php/`
- **Java**: `java/`
- **C#**: `csharp/`
- **Python**: `python/`
- **C++**: `cpp/`

## Usage

Please refer to the code examples in each directory. You will need to replace the API credentials with your own.

## API Documentation

For full API documentation, please visit [https://sdksms.com/api-docs](https://sdksms.com/api-docs).
